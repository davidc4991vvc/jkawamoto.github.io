lastmod: 2014-03-09
date: 2014-03-09
slug: converting-pdf-to-eps
---
```sh
brew install xpdf
pdftops -eps hoge.pdf piyo.eps
```

フォントの問題がある場合は，

```sh
cd /usr/local/ghostscript
sudo ln -s ./9.07/Resource/Font ./font
```

と `/usr/local/etc/xpdfrc` を編集してフォント設定のコメントを有効にする．
